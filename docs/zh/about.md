# 关于我们

## 项目简介

**供应链尽责管理指南系列**是一套面向合规、采购与ESG专业人员的实操参考手册，覆盖供应链尽责管理六大支柱：劳工与社会责任、环境合规、供应链安全、知识产权保护、商业道德与财务韧性、矿产与原材料溯源。

内容基于 OECD 尽责管理指南、UN 指导原则及多项国际合规标准编制，辅以真实案例研究与可直接使用的模板工具，旨在帮助企业以实操方式建立和完善供应链合规管理体系。
