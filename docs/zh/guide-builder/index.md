# 📚 指南生成器 — 概览

本节教您如何使用任意主流 AI（Claude、GPT-4、Gemini 等）为自己的机构**从零生成一份完整的供应链尽职调查审核指南**。

如果您想使用我们预构建的工具自动完成全流程，请下载并运行我们的 **Streamlit 指南生成器应用**（见下方）。

---

## 为什么需要"自己的"指南？

市面上的通用审核指南往往存在三个问题：

| 问题 | 影响 |
|------|------|
| 行业不匹配 | 服装厂的检查表无法直接用于化工供应商 |
| 法规不适用 | 针对欧盟市场的要求不等于您面对 UFLPA/LkSG 的义务 |
| 深度不足 | 缺乏可直接使用的访谈问题、违规示例、整改模板 |

用 AI 生成定制指南，您可以将**您的机构名称、行业、市场、法规框架**直接注入文档，一次性生成可实际使用的审核手册。

---

## 核心方法：文件简介 + 分章生成

```
① 填写文件简介（机构背景、适用法规、核心议题）
        ↓
② 让 AI 生成完整目录（8-12章）
        ↓
③ 确认/调整目录
        ↓
④ 逐章生成（每章 1,500-3,000 字）
        ↓
⑤ 汇总为完整文档，导出 Markdown / Word
```

这种**渐进式生成**方式有两个关键优势：
- **质量控制**：每章生成后可立即审阅，不满意可重新生成
- **上下文管理**：每次只生成一章，避免 AI 因上下文过长而质量下降

## 质量保障原则

生成后请对照以下清单进行人工审核：

- [ ] 所有引用的法规条款是否真实存在（防止 AI 编造）
- [ ] 数字阈值是否符合现行版本（如工时上限、罚款金额）
- [ ] 标准版本号是否为最新（如 SA8000:2014 已有更新吗？）
- [ ] 中国本地法规引用格式是否正确（《法规名称》第X条）
- [ ] 检查表项目是否适用于您的行业和供应商规模

:::warning 免责声明
AI 生成的指南仅作为起点，不构成法律建议。在正式使用前，请由具备资质的合规专业人员进行审核。法规要求随时可能变化，请以官方来源为准。
:::
