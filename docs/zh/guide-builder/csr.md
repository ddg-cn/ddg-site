# 👷 生成 CSR 劳工与社会责任审核指南

## 文件简介模板

将以下简介复制到 AI 对话中，根据您的情况填写 `[ ]` 内的内容：

```
【文件生成简介】
指南类型：CSR 劳工与社会责任审核指南
使用机构：[您的机构名称]
所属行业：[服装/电子/食品/其他]
供应链角色：[品牌商/一级供应商/贸易商]
主要目标市场：[欧盟/美国/英国/中国/其他]
适用法规框架：[LkSG/CSDDD/UFLPA/其他]
供应商总数（约）：[N]家，管理[N]级供应商

【深度与侧重】
面向对象：采购合规团队 + 受审供应商管理层
深度：实操型——须包含具体检查表、访谈问题、审核员现场技巧、常见违规示例
特别关注点：[工人住宿/旺季加班/新疆棉溯源/分包商管理/其他]

【参考标准】
SA8000:2014、SMETA 4-pillar、amfori BSCI、ILO 核心公约、LkSG、CSDDD、UFLPA

【核心议题】
工时与休息日、工资与福利、强迫劳动与童工、结社自由、
职业安全健康、员工住宿、分包管理、申诉机制
```

## 第一步：生成目录

```
你是一名拥有20年经验的供应链尽职调查（DD）专家，专注于帮助中国制造业供应商和跨国采购商建立合规管理体系。

请根据以下文件简介，为《CSR 劳工与社会责任审核指南》生成一份完整、详细的目录结构。

[粘贴您填写好的文件简介]

目录要求：
1. 包含前言、正文各章（8-12章）、附录、延伸阅读
2. 每章下列出3-6个二级节标题
3. 在每章标题后用括号注明该章的核心交付物（例：审核检查表/访谈问题/法规对照表）
4. 目录须覆盖：工时与休息日、工资与福利、强迫劳动与童工、结社自由、职业安全健康、员工住宿、分包管理、申诉机制
5. 参照标准：SA8000:2014、SMETA 4-pillar、amfori BSCI、ILO核心公约 等

请直接输出目录，格式为：
# 第N章　章标题（核心交付物）
## N.1　节标题
## N.2　节标题

不需要解释，直接输出目录。
```

## 第二步：逐章生成

确认目录后，针对每一章发送以下提示词（将 `{章节标题}` 替换为实际章节名）：

```
你正在撰写《CSR 劳工与社会责任审核指南》的第 N/总章数 章。

【文件简介】
[粘贴您的文件简介]

【完整目录】
[粘贴上一步生成的完整目录]

【当前任务】
请完整撰写：{章节标题}

写作要求：
- 深度：每节至少400字，包含具体可操作内容
- 必须包含（适用时）：
  * 审核检查点列表（表格形式）
  * 常见违规示例及整改建议
  * 相关法规/标准原文引用（注明出处）
  * 审核员现场操作技巧
  * 工人或管理层访谈问题（如适用）
- 风格：实操手册，不是学术论文
- 引用标准时注明：标准名称 + 章节号（如：SA8000:2014 §4.3）
- 中国法规引用格式：《法规名称》第X条第X款

请用简体中文撰写本章全文。
请直接开始撰写本章内容，不要输出"好的，我来撰写..."之类的开场白。
```

## 质量检查清单

生成完成后，请逐项核实：

- [ ] SA8000:2014 章节引用格式是否正确（§4.x）
- [ ] 工时上限数字是否与《劳动法》第41条一致（每周不超过44小时正常工时，加班不超过3小时/天、36小时/月）
- [ ] UFLPA "可反驳推定" 的举证责任描述是否准确
- [ ] LkSG §3(2) 对直接和间接供应商的区分是否正确
- [ ] SMETA 4-pillar vs 2-pillar 的区别是否正确描述
- [ ] 工人访谈问题是否覆盖私下访谈技巧（防管理层陪同干扰）
- [ ] 是否包含针对新疆棉/强迫劳动的专项检查方法（如适用）
