# 📋 生成综合尽职调查指南（六柱全覆盖）

本页适用于需要一次性生成**覆盖全部六大支柱**（CSR、环境、安全、IP、道德/财务、矿产溯源）的完整机构定制版尽职调查指南。

如果您只需要单一支柱，请使用左侧菜单中对应的专项提示词页面——更聚焦的提示词通常会产生更高质量的输出。

---

## 文件简介模板

```

使用机构：[您的机构名称]
```

## 第一步：生成综合目录

```
你是一名拥有20年经验的供应链尽职调查（DD）专家，
专注于帮助中国制造业供应商和跨国采购商建立六柱合规管理体系。

请根据以下文件简介，为《供应链综合尽职调查指南》生成一份完整、
结构清晰的目录。

[粘贴您填写好的文件简介]

目录要求：
1. 结构：前言 → 第0章（总论/六柱框架） → 六大支柱（各2-3章）
   → 整合章节（跨柱风险矩阵、整改管理）→ 附录
2. 每章下列出3-5个二级节标题
3. 在每章标题后用括号标注核心交付物
   （例：审核检查表/法规对照表/访谈问题库）
4. 六大支柱章节须包含：
   - 第一支柱：社会责任（CSR）——工时、工资、强迫劳动、OHS、住宿
   - 第二支柱：环境合规——排放、能耗、危废、EUDR/REACH
   - 第三支柱：供应链安全——实物安全、网络安全、人员管理
   - 第四支柱：知识产权——NDA管理、假冒品检测、海关备案
   - 第五支柱：商业道德与财务韧性——反腐败、财务评估、举报机制
   - 第六支柱：矿产溯源——3TG/钴/棉花/棕榈油、CoC文件核查
5. 最后一章（整合章节）须包含：跨柱综合风险矩阵、供应商分级管理、
   整改跟踪（CAP）管理流程

请直接输出目录，格式为：
# 第N章　章标题（核心交付物）
## N.1　节标题
## N.2　节标题

不需要解释，直接输出目录。
```

## 第二步：逐章生成

按第0章 → 各支柱第一章 → 各支柱第二章 → …… → 整合章节 → 附录的顺序逐章生成。
每次生成一章，确认满意后再继续。
:::

```

[粘贴您的文件简介]

【完整目录】
[粘贴第一步生成的完整目录]

【当前任务】
请完整撰写：{章节标题}（属于：{所属支柱/模块}）

写作要求：
- 深度：每节至少400字，包含具体可操作内容
- 必须包含（适用时）：
  * 审核检查点列表（表格形式，含合规标准依据列）
  * 常见违规示例及整改建议
  * 相关法规/标准原文引用（注明标准名称+条款号）
  * 审核员现场操作技巧
  * 访谈问题（工人/管理层，注明访谈对象）
  * 与其他支柱的交叉风险说明（如适用）
- 引用标准格式：标准名称 + 章节号（如：SA8000:2014 §4.3）
- 中国法规格式：《法规名称》第X条第X款
- 风格：实操手册，不是学术论文；可直接用于现场审核

请用简体中文撰写本章全文。
请直接开始撰写，不要输出"好的，我来撰写……"之类的开场白。
```

## 分批生成策略

完整的六柱指南通常包含 14 章 + 4 个附录，建议分三批进行：

**第一批：框架与高优先支柱**
```
请生成以下章节（请依次完成，每章生成完毕后停下等我确认）：
前言 → 第0章 → 第1章 → 第2章（CSR支柱）
```

**第二批：环境与安全支柱**
```
请生成以下章节：
第3章 → 第4章（环境支柱）→ 第5章 → 第6章（安全支柱）
```

**第三批：IP、道德、溯源与整合**
```
请生成以下章节：
第7章 → 第8章（IP支柱）→ 第9章 → 第10章（道德支柱）
→ 第11章 → 第12章（溯源支柱）→ 第13章 → 第14章（整合）
→ 附录A–D
```

## 汇总与导出

全部章节生成完毕后，使用以下提示词汇总并导出：

```
请将以上所有章节内容汇总为一份完整文档。
在文档最开头添加：
1. 封面信息（指南名称、版本号 V1.0、生成日期、机构名称、
   "本文件由AI辅助生成，须经合规专业人员审核后使用"免责声明）
2. 完整目录（含页码估算）
3. 版本变更记录表（预留空格）

输出格式：完整 Markdown，结构清晰，适合导入 Word 或 VitePress。
```

如需导出为 Word 文档，可将生成的 Markdown 内容保存为 `.md` 文件，然后运行：

```bash
pandoc 指南名称.md -o 指南名称.docx --reference-doc=reference-template.docx
```

## 质量检查清单

生成完成后，逐项核实以下内容（重点检查 AI 容易出错的细节）：

**CSR支柱**
- [ ] 工时上限与《劳动法》第41条一致（正常每周≤44h，加班≤3h/天、≤36h/月）
- [ ] UFLPA "可反驳推定" 举证责任描述准确
- [ ] SA8000:2014 章节号格式正确（§4.x）

**环境支柱**
- [ ] EUDR 适用商品清单是否完整（7类：木材、棕榈油、大豆、可可、咖啡、牛肉、橡胶）
- [ ] REACH SVHC 候选清单条目数是否反映当前版本（约240+种）
- [ ] ISO 14001:2015 版本号（非旧版2004）

**安全支柱**
- [ ] C-TPAT 2020版新增要求（供应链安全标准 vs 2003旧版）
- [ ] 中国《网络安全法》《数据安全法》《个人信息保护法》三法引用准确

**IP支柱**
- [ ] TRIPS协议第39条（商业秘密）条款号正确
- [ ] 《中华人民共和国反不正当竞争法》（2019年修正版）相关条款

**道德/财务支柱**
- [ ] FCPA "外国官员"定义涵盖国有企业员工
- [ ] UK Bribery Act 2010 "充分程序" 六项原则完整列出
- [ ] 财务比率基准（流动比率>1.5、速动比率>1.0）已注明行业差异

**矿产溯源支柱**
- [ ] OECD DDG五步法步骤顺序正确
- [ ] CMRT表格版本号是否为最新（当前为v6.x）
- [ ] UFLPA实体清单说明（动态更新，须查阅CBP官网确认最新版本）

**整合章节**
- [ ] 风险矩阵评分规则是否内部一致（无矛盾）
- [ ] CAP整改期限建议是否区分违规等级（零容忍/严重/一般）
