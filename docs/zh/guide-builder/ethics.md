## 质量检查清单

- [ ] FCPA "外国官员"定义是否准确（包含国有企业员工）
- [ ] UK Bribery Act "充分程序" 六项原则是否完整列出
- [ ] 中国《刑法》第164条行贿金额量刑标准是否为最新（2021年修正案调整了起刑点）
- [ ] ISO 37001 认证范围说明是否正确（可单独认证，不依附于ISO 9001）
- [ ] 财务韧性指标：流动比率基准（>1.5）、速动比率（>1.0）等是否说明适用行业局限
- [ ] 举报机制部分是否提及中国劳动法对举报人保护的限制
- [ ] 第三方中间商定义是否明确涵盖经销商、代理商、顾问、政府关系公司
- [ ] OECD 反贿赂公约加入国名单引用是否准确
