4. 目录须覆盖：温室气体与能源、用水管理、废水与废气排放、固废与危废、化学品管理、EUDR毁林合规、碳中和声明

## 第二步：逐章生成

```
你正在撰写《环境合规审核指南》的第 N/总章数 章。

【文件简介】
[粘贴您的文件简介]

【完整目录】
[粘贴目录]

【当前任务】
请完整撰写：{章节标题}

写作要求：
- 深度：每节至少400字，包含具体可操作内容
- 必须包含（适用时）：
  * 审核检查点列表（表格形式）
  * 常见违规示例及整改建议
  * 相关法规/标准原文引用（注明出处）
  * 审核员现场操作技巧（包括文件查阅、现场测量、管理层访谈）
  * 关键监测指标与合规阈值
- 风格：实操手册，引用具体数字和标准版本
- 引用标准注明版本：如 ISO 14001:2015 §6.1.2
- 中国法规引用：如《环境保护法》第X条、《危险废物管理条例》第X条

请用简体中文撰写本章全文。
请直接开始撰写，不要输出开场白。
```

## 质量检查清单

- [ ] EUDR 条例编号是否正确（(EU) 2023/1115）及实施日期（2025年12月）
- [ ] CBAM 涵盖行业是否准确（水泥、钢铁、铝、化肥、电力、氢气）
- [ ] REACH SVHC 候选物质清单说明是否为最新（ECHA 定期更新）
- [ ] ZDHC MRSL 版本是否正确（当前 v3.1）
- [ ] GHG Protocol 范围一/二/三的定义和边界描述是否准确
- [ ] 中国《排污许可证》要求与欧盟 IPPC/IED 的对比是否清晰
- [ ] ISO 14068-1:2023 碳中和五要素是否完整（量化、减少、中和、声明、验证）
- [ ] 危废分类是否引用中国《国家危险废物名录》（2021年版）
