# 资源下载

本页提供尽职调查系列培训材料的下载，涵盖管理层汇报、内部团队培训及供应商导入三类演示文稿。

:::warning 使用前请注意
本页所有可下载文件（演示文稿、模板、问卷等）**仅供参考，不构成法律意见**。文件内容基于公开资料编制，可能不反映最新法规修订。在正式使用前，请由具备资质的合规或法律专业人员进行审核，并以各适用法规的官方原文为准。
:::
