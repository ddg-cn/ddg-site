# 富士康 / 苹果：工人权益与品牌压力（2010–至今）

**当"最严审核"遭遇"最低成本"**

> **关联指南：** [企业社会责任审核指南](/zh/csr-guide) · **适用行业：** 消费电子、代工制造

---

## 背景

富士康科技集团（鸿海精密旗下）是全球最大的电子产品代工厂，iPhone、iPad、MacBook 等苹果核心产品均在其工厂组装。2010年，富士康深圳龙华工厂发生连续工人跳楼事件，引发全球关注，成为供应链劳工问题的标志性事件。

---

## 事件时间线

| 时间 | 事件 |
|------|------|
| 2010年1–11月 | 深圳龙华/观澜工厂发生**18起工人坠楼事件**，其中14人死亡 |
| 2010年5月 | 苹果委托公正劳工协会（FLA）开展审核，富士康提高基本工资30% |
| 2012年3月 | FLA 发布报告：确认存在**过度加班、安全隐患、工资欠付**等问题 |
| 2012年10月 | 郑州工厂爆发工人罢工，起因为质量检验员苛责导致的劳动争议 |
| 2014年 | 成都和郑州工厂被记录仍存在周工时超过60小时的情况 |
| 2019年 | 中国劳工通讯与澳大利亚战略政策研究所报告显示加班问题持续 |
| 2022年11月 | 郑州工厂（"iPhone城"）爆发大规模工人骚乱，起因为防疫封控与奖金纠纷 |

## 审核局限性的典型案例

苹果是业内对供应商审核投入最大的品牌之一——每年发布《供应商责任进展报告》，对数百家工厂进行审核。然而，问题为何持续？

### 问题根源

**1. 结构性矛盾未解决**

苹果的季节性订单峰谷落差极大（尤其是 iPhone 发布窗口），要求富士康在短时间内大幅扩充产能。这导致：
- 工厂需要大量**临时工/劳务派遣工**（在中国，这类工人劳动保护较弱）
- 旺季加班时长系统性超标，即便工人"自愿"，也受到隐性压力

**2. 审核时机与频率**

- 审核通常提前安排，工厂有时间"整理"出勤记录
- 旺季（需求最大、违规最多时）反而是审核空窗期
- FLA 2012年报告显示：**超过50%的受访工人表示曾加班超过法定上限**，而工厂仍通过了苹果的审核

**3. 工人声音渠道失效**

- 工厂内部设有投诉热线，但工人普遍不信任匿名性
- 独立工会在中国受到法律限制
- 研究发现，工人在接受审核员访谈时，有时被管理层提前"辅导"

**4. 自杀事件的深层原因**

2010年事件后，富士康在宿舍楼加装防坠网，被媒体称为"自杀防护网"——这一应急措施本身说明问题在于**系统性压力**（高强度流水线、宿舍式管理、与家乡的社会隔绝），而非个案心理问题。

## 企业与行业的回应

### 苹果的措施

- 加入 FLA（公正劳工协会），接受第三方监督
- 发布年度《供应商责任进展报告》（2007年至今）
- 要求富士康将月基本工资提升至不低于当地最低工资的1.5倍
- 推出**供应商员工教育计划**（免费大学课程、职业培训）
- 2022年承诺在供应链中推广**"负责任招募"**，消除招募费（以减少债务束缚）

### 富士康的措施

- 多次上调基本工资（2010、2012、2018年均有调薪）
- 引入机器人生产线（部分替代重复性流水线岗位）
- 加强工人援助计划（EAP）与心理辅导
- 在郑州、成都等内陆城市建设新厂，使工人离家更近

## 对审核实践的启示

::: tip 关键审核问题

**在电子代工供应商的工人访谈中，务必询问：**

1. 您本月实际工作了多少小时？旺季呢？（与工资单交叉核验）
2. 您是正式员工还是劳务派遣/临时工？派遣比例是否超过10%（中国法规上限）？
3. 如果您因健康或个人原因拒绝加班，会有什么后果？
4. 宿舍/食堂费用是从工资中扣除的吗？扣除额是否合理透明？
5. 您知道厂内有投诉渠道吗？您会使用它吗？为什么？

**文件审核重点：**
- 比对旺季（Q3-Q4）与淡季的出勤记录，识别系统性超时规律
- 核查劳务派遣合同比例（中国《劳务派遣暂行规定》上限10%）
- 要求工资单原件，核算隐性费用（餐费、住宿费、工装费扣除后的实得工资）

:::
