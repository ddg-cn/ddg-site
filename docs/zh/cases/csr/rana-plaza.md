# Rana Plaza 大楼坍塌（2013）

**快时尚供应链的最高代价**

> **关联指南：** [企业社会责任审核指南](/zh/csr-guide) · **适用行业：** 服装、纺织、鞋类

---

## 事件经过

2013年4月24日，孟加拉国达卡郊区的 Rana Plaza 商业综合楼轰然倒塌，造成**1,134人死亡、2,500余人受伤**，是人类历史上伤亡最惨重的工业事故之一。

大楼共八层，其中五至八层为服装工厂，为 Primark、Benetton、Mango、Walmart 等欧美品牌加工服装。事发前一天，大楼已出现明显裂缝，银行等商业租户已宣布停业疏散，但工厂管理者以"赶订单"为由，**强迫工人返岗上班**，并威胁不上班者扣薪。次日清晨，大楼在工人上班后数小时内整体垮塌。

### 结构性原因

- 大楼原批准用途为商业/零售，**未经许可擅自加盖四层**作为工厂
- 大楼建设使用了劣质材料，设计承重远不足以支撑重型工业设备
- 数台大型发电机（工厂自备电源）安装于顶层，振动加速了结构疲劳

---

## 供应链审核的失败

Rana Plaza 揭示了当时"合规审核"体系的系统性缺陷：

| 失败环节 | 具体表现 |
|---------|---------|
| **分包隐瞒** | 品牌采购商只知道一级供应商，工厂秘密将订单转包至 Rana Plaza，未经披露 |
| **审核频率不足** | 多数品牌每年仅审核一次，且提前告知，工厂有充裕时间"布景" |
| **建筑安全盲区** | 审核清单聚焦劳工标准（加班、工资），**从不检查建筑结构安全** |
| **审核员技能局限** | SA8000、BSCI等社会审核员没有工程背景，无法识别结构风险 |
| **买方压力传导** | 极短的交期与极低的采购价迫使工厂压缩成本，包括安全投入 |

## 后续响应

### 孟加拉国《服装业协议》（Accord on Fire and Building Safety）

事故发生后数周，220余个品牌签署了由工会主导的**《孟加拉国消防与建筑安全协议》**（Accord），核心条款：

- 委托独立工程师对工厂进行**建筑、电气、消防**三项专项检查
- 检查结果**公开披露**（品牌无权隐瞒不合格工厂）
- 品牌承诺为整改提供**资金支持**（不能以"供应商费用"转嫁工厂）
- 设立工人**投诉热线**，举报无须担心被解雇

截至2018年，Accord 覆盖超过1,600家工厂，发现并整改了10万余项安全隐患。

### Rana Plaza 赔偿基金

国际劳工组织（ILO）主导设立赔偿基金，最终募集约3,000万美元。Primark 等品牌个别捐款，但大量品牌在初期**拒绝承认在该楼采购**，以逃避赔偿责任，品牌声誉受到严重冲击。

## 对审核实践的启示

::: tip 审核人员应特别关注

**建筑安全检查须成为标准环节：**
- 核查建筑使用许可证与实际用途是否一致
- 查看是否有超出批准层数的违规加建
- 重型设备（发电机、空调机组）安装是否有结构评估报告

**分包管控须列入合同条款：**
- 供应商合同须明确"未经书面批准不得分包"
- 定期抽查订单实际生产地点（可通过布料卷标、缝制标签追溯）
- 要求供应商披露所有二级加工厂并纳入审核范围

**工人声音渠道：**
- 撤销品牌LOGO等标识，以匿名方式开展工人访谈
- 设立独立举报渠道（不通过工厂管理层转达）
- 询问"如果今天上班有安全顾虑，你有权拒绝吗？"

:::
