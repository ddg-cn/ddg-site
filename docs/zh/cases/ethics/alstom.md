# 阿尔斯通《美国陷阱》：FCPA的猎捕（2014）

**当外国企业高管成为美国司法的筹码**

> **关联指南：** [商业道德、反贿赂与财务韧性指南](/zh/ethics-guide) · **适用行业：** 能源基础设施、工程承包

---

## 事件经过

2013年4月，法国工业巨头阿尔斯通（Alstom）美国分公司高管弗雷德里克·皮耶鲁奇（Frédéric Pierucci）在纽约肯尼迪机场被FBI逮捕。指控内容：参与向印度尼西亚官员行贿，以换取塔拉汉（Tarahan）燃煤电厂合同。

这一事件被皮耶鲁奇本人记录在其畅销书《美国陷阱》（*The American Trap*）中，揭示了美国《境外反腐败法》（FCPA）如何被用于打击竞争对手的地缘政治工具。

### 行贿规模与路径

阿尔斯通在印尼、沙特、埃及、巴哈马等多国的项目中，通过"顾问费"渠道向当地官员行贿，累计金额超过**7,500万美元**。行贿路径：

```
阿尔斯通 → 空壳顾问公司（注册于避税地） → 当地官员
```

顾问公司提供虚假"咨询服务"，实际作用是充当行贿中间人。

## 法律结果

| 主体 | 结果 |
|------|------|
| 阿尔斯通美国分公司 | 认罪，支付**7.72亿美元**罚款（2014年，时为FCPA史上最高罚款之一） |
| 皮耶鲁奇本人 | 在美国联邦监狱服刑30个月 |
| 阿尔斯通集团 | 在调查期间（2014年），将能源业务出售给通用电气（GE），售价约为市场估值的低端 |

### 《美国陷阱》的争议叙事

皮耶鲁奇认为：美国司法部有意在阿尔斯通最脆弱的时候（调查进行中）向法国政府施压，迫使法方同意将阿尔斯通能源业务出售给通用电气——这是一项对美国国家利益高度战略性的收购。

无论这一叙述是否完整，它揭示了一个关键事实：**FCPA的域外管辖权极为宽泛**，只要涉及美元结算、美国银行系统、美国公民或美国上市公司，美国司法部即可主张管辖。

## 对供应链尽职调查的启示

::: tip 第三方顾问管理是FCPA合规的重灾区

阿尔斯通案（以及大多数FCPA大案）的核心都是**第三方顾问（agents/intermediaries）**。以下是实务中的关键控制点：

**尽职调查触发条件：**
- 顾问在政府项目国家有关联背景
- 顾问费率远高于市场水平（"成功费"超过合同额5%应警惕）
- 顾问坚持通过空壳公司或第三国账户收款
- 顾问无法说明其"服务内容"的具体价值

**合同控制条款（必备）：**
- FCPA/UKBA/相关法律合规声明（representations & warranties）
- 审计权条款（有权审计顾问的费用记录）
- 单边终止权（发现违规可立即终止合同）
- 禁止转包条款（顾问不得将职责转委托给第四方）

**支付管控：**
- 所有顾问费须经财务合规审批（双人签字）
- 禁止现金支付、禁止向第三方账户付款
- 付款须与可证明的服务输出挂钩，而非"里程碑"

:::
