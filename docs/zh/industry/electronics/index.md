# 消费电子与代工行业专项模板

> **行业背景：** 消费电子供应链层级深、地域广，代工（OEM/ODM/EMS）模式普遍。主要风险：冲突矿产（3TG+钴）、数据安全与知识产权、劳工（大量劳务派遣）、环境（WEEE/RoHS）。关键参照：EICC（现RBA）行业准则、苹果/华为/三星供应商行为准则。

## 行业风险优先级矩阵

| 风险类别 | 风险等级 | 核心指标 |
|---------|---------|---------|
| 冲突矿产（3TG：锡/钽/钨/金 + 钴） | 🔴 极高 | CMRT/EMRT填报、冶炼厂溯源 |
| 数据安全与客户IP | 🔴 极高 | 数据访问控制、员工保密协议 |
| 劳务派遣比例过高 | 🟠 高 | 派遣工比例、同工同酬 |
| 过度加班（旺季） | 🟠 高 | 旺季工时记录 |
| RoHS / WEEE 合规 | 🟠 高 | 有害物质检测、回收体系 |
| 供应商知识产权保护 | 🟠 高 | NDA覆盖、访问控制 |
| 网络安全（供应链攻击） | 🟠 高 | 软件更新管控、第三方访问 |
| 有害物质（焊接/喷涂） | 🟡 中高 | 职业健康监测、个人防护 |

## 市场准入认证

> 以下认证是进入主要目标市场的**强制性或事实标准要求**，与工厂合规审核并行，买家尽职调查时须核验其有效性。

### CE 合格标志（欧盟）

**CE 标志**是向欧盟/欧洲经济区市场销售产品的强制性合格声明，覆盖电子产品、机械、玩具、医疗器械、无线电设备、低压设备等几乎所有消费电子品类。

| 相关指令/法规 | 适用品类 | 核心要求 |
|------------|---------|---------|
| **RED（无线电设备指令 2014/53/EU）** | 含无线模块的设备（WiFi、BT、蜂窝） | 型式审查、技术文件、DoC声明 |
| **LVD（低压指令 2014/35/EU）** | 50V–1000V交流/75V–1500V直流设备 | 安全测试、EU公告机构审查 |
| **EMC 指令（2014/30/EU）** | 所有电气/电子设备 | 电磁兼容测试 |
| **RoHS（2011/65/EU + 修订）** | 电子电气设备 | 10种有害物质限量 |
| **Ecodesign（EU 2019/2021）** | 能源相关产品（SRP，2021年后分阶段） | 能效、可修复性、可回收性要求 |

::: warning 审核员提示
CE 标志不是由政府机构颁发——制造商自我声明合格。审核时须索取**技术文件（Technical File）**和**合格声明（DoC）**，而非仅要求展示标志。无 DoC 或技术文件不完整，视为不合规。
:::

### FCC 认证（美国）

美国**联邦通信委员会（FCC）**对所有在美销售的无线电频率发射设备实施强制认证：

| 授权类别 | 适用设备 | 程序 |
|---------|---------|------|
| **FCC ID 认证** | 故意辐射体（WiFi路由器、手机、蓝牙设备） | 须通过FCC认可实验室测试，在FCC数据库注册获取FCC ID |
| **符合性声明（SDoC）** | 非故意辐射体（PC、显示器、充电器） | 制造商自测，保留技术文件 |
| **豁免** | 极低功率设备 | 须符合 Part 15 限值 |

::: tip 实操建议
向美国供货的电子代工厂应在审核前提供产品的FCC ID（可在 fcc.gov/oet/ea/fccid 核验真实性）。FCC ID伪造是知识产权高风险信号。
:::

### EN 71（欧盟玩具安全指令）

EN 71 是欧盟玩具安全标准系列，对含玩具属性的电子产品（益智玩具、儿童平板、遥控玩具）具有强制适用性：

| 标准 | 内容 |
|------|------|
| EN 71-1 | 机械和物理性能（尖锐边缘、窒息风险） |
| EN 71-2 | 燃烧性能 |
| EN 71-3 | 特定元素迁移（重金属）|

14岁以下儿童使用的电子产品须同时满足 CE/LVD/RED 和 EN 71。

### BIS 认证（印度）

**印度标准局（BIS）**对多类电子产品实施强制认证（CRS 计划），进口至印度须在 BIS 注册：

- **强制品类**：手机、笔记本电脑、平板、LED灯、电源适配器、智能手表、机顶盒等（品类持续扩增）
- **注册方式**：境外制造商须通过印度授权代表（AIR）向 BIS 申请 R 号注册
- **不合规后果**：货物在印度口岸被扣押，无 BIS 注册的同类产品进口被全面禁止

::: info 中国出口商注意
BIS 强制认证产品目录扩展速度较快（2022–2024年新增 20+ 品类）。向印度出口前须在 bis.gov.in 核查最新目录。
:::
