# 化工与危险品行业专项模板

> **行业背景：** 化工供应商是供应链中监管最密集的环节之一，面临 REACH / GHS / CLP / 危险废物 / 职业健康等多重合规义务。在反腐败领域，工程承包和政府项目资质申请是高风险场景。双重用途化学品（可转化为武器前体物）还涉及出口管制。

## 行业风险优先级矩阵

| 风险类别 | 风险等级 | 核心指标 |
|---------|---------|---------|
| 危险废物非法处置 | 🔴 极高 | 转移联单、处置单位资质 |
| 有毒气体泄漏/爆炸 | 🔴 极高 | PSM（过程安全管理）体系 |
| REACH/SVHCs合规 | 🟠 高 | 物质注册、SVHC披露 |
| 职业健康（化学品暴露） | 🟠 高 | 职业卫生监测、PPE合规 |
| 出口管制（双重用途） | 🟠 高 | 最终用户核查、出口许可证 |
| 废水/废气排放 | 🟠 高 | 在线监测数据、超标历史 |
| 反腐败（工程/资质） | 🟡 中高 | 第三方顾问、政府关系 |

## 关键法规参照

| 法规 / 标准 | 适用场景 |
|------------|---------|
| **REACH（欧盟化学品注册）** | 欧盟市场出口 |
| **CLP / GHS** | 化学品分类标签 |
| **Process Safety Management（PSM，OSHA）** | 高危化工设施 |
| **《危险化学品安全管理条例》（中国）** | 中国境内 |
| **《固体废物污染环境防治法》** | 危废处置 |
| **EAR / ITAR** | 双重用途化学品出口（美国管制） |

| **BIS 认证（印度）** | 危险化学品包装、工业气体、农药（印度强制品类） |

## 市场准入认证

### NSF 认证（水处理与工业化学品）

**NSF International** 对与饮用水接触的化学品和材料设有强制性认证要求，是进入美国及加拿大市场的实质门槛：

| 标准 | 适用范围 | 市场要求 |
|------|---------|---------|
| **NSF/ANSI 60** | 直接加入饮用水的水处理化学品（氯、絮凝剂、阻垢剂等） | 美国各州对公共水务强制要求；私人水厂采购的事实标准 |
| **NSF/ANSI 61** | 与饮用水间接接触的材料（管件、涂层、密封件） | 美国大多数州及加拿大法规强制 |
| **NSF/ANSI 50** | 游泳池/水疗水处理化学品 | 美国游泳池化学品市场事实标准 |

::: tip
出口水处理化学品至美国的中国企业，须由**供应商或美国进口商**持有对应 NSF 认证，无认证产品无法进入市政水务采购名单。
:::

### BIS 认证（印度）

印度对以下化工品类实施 BIS 强制认证：

- **农药原药及制剂**：须符合 IS 标准并通过 BIS 许可（配合农药注册）
- **工业气体（高压容器）**：气瓶须符合 IS 7285，制造商须持 BIS 许可
- **危险化学品包装**：出口至印度的危化品须符合联合国 GHS/CLP 标签要求，并附印地语安全数据表（SDS）
