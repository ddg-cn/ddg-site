## 模块 CH1　过程安全（PSM）

| # | 问题 | 是 | 否 | 部分 | 备注 |
|---|------|----|----|------|------|
| CH1.1 | 是否建立了过程安全管理（PSM）体系，覆盖所有高危工艺？ | | | | |
| CH1.2 | 是否定期开展危险与可操作性研究（HAZOP）？ | | | | |
| CH1.3 | 是否有书面的紧急响应预案（含泄漏、爆炸、火灾场景）？ | | | | |
| CH1.4 | 过去5年内是否发生过重大工艺事故（泄漏、火灾、爆炸）？ | | | | |
| CH1.5 | 承压设备是否按期检验，有效期内？ | | | | |

## 模块 CH2　危险废物管理

|---|------|----|----|------|------|
