# 行业专项模板库

通用模板提供框架，但不同行业的风险画像差异显著——服装厂关注工时与童工，化工企业关注危废排放，软件供应商关注数据安全与知识产权。

本库按行业提供深度定制的评估工具，每套工具均包含：行业专项SAQ、审核前文件清单、关键访谈问题，以及该行业历史案例参照。

## 可用行业模板

| 行业 | 主要风险领域 | 完成度 |
|------|------------|--------|
| [服装与纺织](./garment/) | 劳工（加班/童工）、化学品（REACH）、分包隐瞒 | ✅ 完整 |
| [消费电子与代工](./electronics/) | 矿产溯源（冲突矿产）、数据安全、劳工 | ✅ 完整 |
| [食品与农业](./food/) | 食品安全、农业劳工、用水管理、EUDR | ✅ 完整 |
| [化工与危险品](./chemicals/) | 危废管理、REACH/GHS、职业健康 | ✅ 完整 |
| [建筑与工程](./construction/) | 建筑安全、分包层级、反腐败 | ✅ 完整 |

## 如何选择模板

```
供应商类型？
├─ 生产型（工厂）→ 选择对应行业 SAQ
├─ 贸易商/中间商 → 使用通用 SAQ Section A+B
└─ 服务型（IT/物流/咨询）→ 使用通用 SAQ Section A+G

风险等级？
├─ 高风险（高关注度地区 + 核心供应商）→ 完整现场审核 + 全量SAQ
├─ 中风险 → 文件审核 + 重点章节SAQ
└─ 低风险 → SAQ自评 + 抽查
```

## 与通用模板的关系

行业专项模板**补充**通用模板，不替代。推荐使用方式：

1. **通用 SAQ**（[Section A–H](/zh/templates/saq)）作为基础筛查
2. 根据供应商行业叠加对应的**行业专项模块**
3. 审核前文件请求使用**行业专项清单**替代通用清单
