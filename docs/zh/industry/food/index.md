# 食品与农业行业专项模板

> **行业背景：** 食品供应链从农场到零售终端，层级多、溯源难，风险涵盖食品安全（FSMA/ISO 22000）、农业劳工（季节性用工、儿童劳动）、用水与土地冲突，以及2023年起生效的《欧盟毁林法规》（EUDR）对大豆、棕榈油、可可等商品的溯源要求。

## 行业风险优先级矩阵

| 风险类别 | 风险等级 | 核心指标 |
|---------|---------|---------|
| 食品安全 / 掺假 | 🔴 极高 | HACCP认证、召回历史 |
| 农业劳工（儿童/强迫） | 🔴 极高 | 季节性用工、招募费 |
| EUDR合规（毁林） | 🔴 极高（欧洲市场）| 农场GPS坐标、土地权属 |
| **动物福利** | 🔴 极高（畜牧/水产/乳品）| 五大自由、禁用笼养/强制填饲、OIE人道屠宰 |
| 用水管理 | 🟠 高 | 水资源压力地区的用水许可 |
| **生物多样性/KBA** | 🟠 高 | 设施是否位于KBA或保护区缓冲区 |
| 农药残留 | 🟠 高 | 超标历史、检测频率 |
| 土地冲突/FPIC | 🟠 高 | 原住民权利、社区申诉机制 |
| 冷链与物流安全 | 🟡 中 | 温度记录、防篡改 |
| 包装材料合规 | 🟡 中 | 食品接触材料法规 |

## 市场准入认证

> 以下为主要出口目标市场的强制性或实质性准入认证，买家须在采购合同中明确，并在尽职调查中核验有效性。

### FDA 与 FSMA（美国食品安全现代化法）

**美国 FDA** 对食品进口实施预防性监管，主要义务包括：

| 制度 | 适用方 | 核心义务 |
|------|--------|---------|
| **FSVP（外国供应商验证计划）** | 美国进口商 | 须对外国供应商进行危害分析和年度绩效核查 |
| **PCQI（预防控制合格人员）** | 在美登记的食品设施 | 食品安全计划须由 PCQI 认证人员签署 |
| **设施注册** | 向美国出口的食品制造/加工/包装/储存设施 | 须向 FDA 注册（每两年续期），并授权 FDA 检查 |
| **生物恐怖主义法规（PTF）** | 高风险食品（婴幼儿配方、软奶酪、烟熏海鲜等）| 提前通知（Advance Notice），最长72小时 |

::: warning FSMA 对中国供应商的直接影响
向美国出口的中国食品工厂须在 FDA 注册，且美国进口商对其承担 FSVP 验证义务。未注册或拒绝检查可导致货物在美国口岸被自动扣押。
:::

### NSF 认证（美国/国际）

**NSF International** 提供独立认证，在以下场景与食品供应链直接相关：

| 认证 | 适用产品/场景 | 要求 |
|------|------------|------|
| **NSF/ANSI 51** | 食品设备与材料（食品接触面） | 用于餐饮业和食品加工设备采购 |
| **NSF/ANSI 61** | 饮用水系统（管道、阀门、储罐） | 美国各州对公共饮用水系统材料强制要求 |
| **NSF/ANSI 173** | 膳食补充剂（GMP合规） | 美国进口商对补充剂制造商的常见要求 |
| **NSF 食品安全认证** | 食品加工设施 GMP | 替代性 GFSI 认证（部分北美买家接受） |

### BIS 认证（印度）

印度市场对进口食品的主要强制要求：

- **FSSAI 注册/许可**：所有向印度出口的食品制造商须向印度食品安全与标准局（FSSAI）申请进口商注册，部分类别须取得制造商认证（Central License）
- **强制品类**：乳制品、食用油、天然矿泉水、有机食品、营养强化食品须通过 BIS 品类认证（IS标准对应）
- **标签要求**：须标注 FSSAI 许可证号、净含量（公制单位）、最佳食用日期
