# 建筑与工程行业专项模板

> **行业背景：** 建筑与工程承包行业以多层分包为常态，是反腐败合规的高风险领域（政府项目、规划许可、检验验收）。劳工方面以短期合同工和外来务工人员为主，工伤率高。环境层面涉及建筑废料、粉尘、噪音和土壤污染。

## 行业风险优先级矩阵

| 风险类别 | 风险等级 | 核心指标 |
|---------|---------|---------|
| 反腐败（规划/资质/验收） | 🔴 极高 | 政府关系费用、第三方顾问 |
| 施工安全 / 高空作业 | 🔴 极高 | 工伤记录、安全培训、脚手架检查 |
| 分包层级不透明 | 🟠 高 | 分包商名单、资质核验 |
| 外来务工人员工资欠付 | 🟠 高 | 工资发放记录、农民工保证金 |
| 建筑废料非法处置 | 🟠 高 | 渣土运输许可、消纳场资质 |
| 材料造假（钢筋、混凝土强度） | 🟠 高 | 第三方检测报告、批次溯源 |
| 安全资质挂靠 | 🟡 中高 | 人员证件与实际在岗核验 |

## 关键法规参照

| 法规 / 标准 | 适用场景 |
|------------|---------|
| **《建筑法》《安全生产法》（中国）** | 境内工程 |
| **ISO 45001（职业健康与安全）** | 建筑施工 |
| **OHSAS 18001（旧标准，仍有参照）** | 过渡期 |
| **《建设工程质量管理条例》** | 材料与施工质量 |
| **UK Bribery Act / FCPA** | 海外工程反腐败 |
| **《保障农民工工资支付条例》** | 中国境内农民工工资 |
