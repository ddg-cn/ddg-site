# 服装与纺织行业专项模板

> **行业背景：** 全球服装供应链以劳动密集型生产为主，高度集中于南亚（孟加拉国、印度、巴基斯坦）、东南亚（越南、柬埔寨、缅甸）和中国。该行业历史上存在严重的劳工风险（Rana Plaza）、化学品合规问题（有毒染料）和分包不透明问题。

## 市场准入认证

> 以下认证是进入主要出口目标市场的强制或实质性要求，须在采购合同中明确，并在审核时核验。

### STeP by OEKO-TEX（可持续纺织生产认证）

**STeP**（Sustainable Textile and Leather Production）是 OEKO-TEX 体系中针对**工厂**的综合认证，与 OEKO-TEX Standard 100（产品检测）互补：

| 模块 | 评估内容 |
|------|---------|
| 化学品管理 | MRSL 合规、化学品储存、废水处理 |
| 环境绩效 | 能耗、用水、废物、碳排放 |
| 环境管理 | ISO 14001 体系或等效 |
| 社会责任 | 工时、工资、工会权利、职业安全 |
| 质量管理 | 过程控制、不合格品管理 |
| 安全与健康 | 作业安全、个人防护 |

::: tip
STeP 认证对同时向欧洲快时尚和奢侈品牌供货的工厂特别有价值——H&M、Zara、Hugo Boss 等均接受 STeP 作为工厂合规证明，可减少重复审核。
:::

### CE 标志 — 儿童服装与 PPE

服装行业与 CE 标志的交集主要在两类产品：

- **个人防护装备（PPE，EU 2016/425）**：工作手套、防护背心、防化学品围裙等须强制加贴 CE 标志，须经公告机构（Notified Body）型式检验
- **儿童服装附件含危险物质或绳带**：须符合 EN 14682（儿童服装绳带安全）和 EN 71 玩具安全要求（若附件有玩具属性）

### BIS 认证（印度）

印度市场对纺织品进口有以下强制要求：

- **纺织品标签**：《纺织品（消费者保护）规则》要求标注纤维成分、原产国、制造商信息（中文标签须附英文/印地语）
- **儿童服装**：BIS 已发布儿童服装安全标准（IS 17593），拟强制实施
- **有害物质**：印度 BIS 标准 IS 11871 限制偶氮染料，与欧盟 REACH 附件 XVII 部分重合
