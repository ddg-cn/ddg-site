## 第一部分　劳工与工作条件

### A1　工时与加班

| # | 问题 | 是 | 否 | 部分 | 备注 |
|---|------|----|----|------|------|
| A1.1 | 普通周（非旺季）平均工时是否≤60小时？ | | | | |
| A1.2 | 所有加班是否为工人自愿，且有书面/电子记录？ | | | | |
| A1.3 | 工人是否有权在不受处罚的情况下拒绝加班？ | | | | |
| A1.4 | 连续工作7天后，工人是否获得至少1天休息？ | | | | |
| A1.5 | 旺季最高工时记录是否≤每月288小时（含加班）？ | | | | |
| A1.6 | 工时记录是否为原始电子打卡数据（非手工修改）？ | | | | |

### A2　工资与福利

| # | 问题 | 是 | 否 | 部分 | 备注 |
|---|------|----|----|------|------|
| A2.1 | 基本工资是否≥当地法定最低工资？ | | | | |
| A2.2 | 加班费是否按法定倍率支付（平日1.5×、周末2×、法定节假日3×）？ | | | | |
| A2.3 | 工资是否按时足额发放，无不合理扣押或拖欠？ | | | | |
| A2.4 | 工资单是否以工人可理解的语言提供，列明各项明细？ | | | | |
| A2.5 | 是否为所有正式员工缴纳法定社会保险（养老、医疗、工伤等）？ | | | | |
| A2.6 | 从工资中扣除的餐费、住宿费是否有事先书面告知？ | | | | |

| # | 问题 | 是 | 否 | 部分 | 备注 |
|---|------|----|----|------|------|

| A3.2 | 工人的身份证件（护照、身份证等）是否由工人本人保管？ | | | | |
