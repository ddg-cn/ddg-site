# 供应商 / 受审方资源包

本页汇总**供应商（受审方）**在应对客户尽职调查要求、准备审核及建立合规体系时需要的全部参考资料。

---

## 我需要做什么？

::: info 快速入门指引
1. **阅读并签署**客户发来的《供应商行为准则（SCoC）》
2. **填写**供应商自评问卷（SAQ），如实填报现状
3. **整理文件**，按文件清单准备审核前所需材料
4. **配合现场审核**，包括工厂参观、文件核查、工人访谈
5. **制定纠正行动计划（CAP）**，处理审核发现项
:::

## 合规体系建设建议

::: tip 从零开始建立合规管理体系
1. **任命合规联络人**（可由HR或HSE兼任）——有专人负责是通过审核的首要条件
2. **建立文件管理系统**：合同、工资单、工时记录、培训记录须至少保存3年
3. **张贴必要公告**：工人申诉渠道、安全规程、紧急疏散图——审核员会核查
4. **提前做内审**：参照文件清单，在正式审核前1个月自查一遍
5. **诚实填写SAQ**：自评中掩盖问题，审核时被发现会导致更严重的评级下降
:::
