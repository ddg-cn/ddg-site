# 审核方工具包

本页汇总供**审核员、合规经理及采购方**在供应商尽职调查工作中使用的全部模板与工具。

---

## 核心工具

| 工具 | 说明 | 链接 |
|------|------|------|
| **审核前文件请求清单** | 向供应商索取文件的标准化清单，涵盖六大合规领域 | [前往](/zh/templates/audit-checklist) |
| **访谈问题库** | 工人访谈、管理层访谈的结构化题库，含敏感话题技巧 | [前往](/zh/templates/interview-questions) |
| **操作表单** | 开场/闭幕会议记录、发现项记录、证据追踪表 | [前往](/zh/templates/forms) |
| **政策模板** | 可直接采用的采购政策、供应商合规政策范本 | [前往](/zh/templates/policies) |

## 供应商行为准则（发布给供应商使用）

**[供应商行为准则（SCoC）](/zh/templates/supplier-code)** — 可直接发布给供应商签署的完整行为准则模板，涵盖劳工、环境、安全、知识产权、道德与数据保护等所有条款。

---

## 供应商自评问卷（SAQ）

**[通用SAQ](/zh/templates/saq)** — 可发送给供应商填写的自评问卷，可作为年度风险筛查工具或审核前预评估工具。

::: tip 最佳实践
发送SAQ时建议说明：（1）填写截止日期；（2）数据将如何保密处理；（3）SAQ得分如何影响审核频率或供应商级别评定。
:::
