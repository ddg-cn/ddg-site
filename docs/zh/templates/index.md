# 模板与工具总览

本节收录了供应链六大尽职调查支柱所需的实用文件模板，涵盖政策、问卷、清单、访谈问题库和操作表单。所有模板均需根据企业实际情况、所在司法管辖区法律要求及客户标准进行调整，**不构成法律建议**。

::: tip 使用说明
- 模板中 `[方括号]` 内的内容为需要根据实际情况填写或替换的占位符
- 相关标准与法规随时更新，使用前请核实当前有效版本
- 涉及重大法律义务的文件（如政策条款、合同附件）建议在法律顾问指导下定稿
- 所有问卷和清单均可拆分为单独文件或集成至供应商门户系统使用
:::

| 文件 | 主要用途 | 所涉支柱 | 典型使用方 |
| --- | --- | --- | --- |

| [访谈问题库](./interview-questions) | 工人保密访谈；管理层访谈；各专项审核访谈 | CSR、Ethics、Security、IP | 审核员 |
| [操作表单](./forms) | 日常合规运营所需的填报表单 | Ethics、DD | 采购/合规/员工 |
| [政策模板](./policies) | 建立内部合规政策体系；对供应商展示合规承诺 | Ethics为主，跨支柱 | 合规/法务/管理层 |

## 快速导航：按场景选择模板

**新供应商准入时：**
→ [供应商行为准则](./supplier-code)（要求签署后附于合同）
→ [SAQ](./saq)（要求填写A+B+D节，高风险填写全部）

**年度常规评估时：**
→ [SAQ](./saq)（每年更新，低风险供应商可简化填写B+D+G节）

**现场审核前30天：**
→ [审核前文件请求清单](./audit-checklist)（按审核类型选择对应清单）
→ [访谈问题库](./interview-questions)（按访谈对象选择对应问题）

**发现违规问题后：**
→ [操作表单](./forms) → 整改行动计划（CAP）模板

**建立内部合规政策体系时：**
→ [政策模板](./policies)（反贿赂、礼品招待、利益冲突、举报人保护）
